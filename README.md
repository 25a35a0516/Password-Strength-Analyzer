# 🔐 Smart Password Strength Analyzer

A simple and interactive web application that analyzes the strength of a password in real-time. This project is designed using **Design Thinking principles** to improve user awareness about secure password creation.

---

## 📌 Problem Statement

Many users create weak passwords, making their accounts vulnerable to security threats. There is a need for a simple tool that provides instant feedback on password strength.

---

## 💡 Solution

This project provides a **real-time password strength analyzer** that evaluates user input based on multiple security criteria and gives visual feedback.

---

## 🚀 Features

* 🔍 Real-time password strength checking
* 📊 Visual strength indicator (Weak / Medium / Strong)
* 👁️ Show / Hide password option
* ✅ Validation for:

  * Minimum length (8 characters)
  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* 🎨 Clean and modern user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Regex)

---

## 🧠 Design Thinking Approach

* **Empathize:** Users struggle to create strong passwords
* **Define:** Need for an easy-to-use password validation tool
* **Ideate:** Develop a real-time password analyzer
* **Prototype:** Built using HTML, CSS, and JavaScript
* **Test:** Tested with multiple password inputs

---

## 📷 Demo

Try different passwords like:

* `abc` → Weak

1. Download or clone this repository
2. Open the `index.html` file in any web browser
3. Start typing a password to see live results
